Demo Assets

Believable material for the live builds — so you can point Claude at something that feels like real AATF work without touching anything sensitive.

⚠️ Fictional demo data — not real AATF information

Every file here is invented. AATF's real programme names are used for realism (TELA Maize, PBR cowpea, Striga control, cassava), but all numbers, people, donors, quotes, and commitments are fictional. Placeholder names like "Dr. A. Example" and "Donor Foundation X" are deliberate. Never use this content in real AATF communications.
