# Working with Modern AI — AATF Workpack

A one-day, hands-on AI training workpack prepared for the **African Agricultural Technology Foundation (AATF)**, in partnership with the **Forestry & Agricultural Biotechnology Institute (FABI), University of Pretoria**.

Nairobi · 15 July 2026 · Presented by Prof. Bernard Slippers & Arné Schreuder.

## 🌐 Live site

Once GitHub Pages is enabled, the workpack is hosted at:

**https://neurulation.github.io/aatf-ai-workshop/**

Everything is self-contained static HTML — it also works fully offline. Just open `index.html` in a browser.

## Design

A single, offline-first design system built around **FABI green** and **AATF harvest gold** — no external dependencies, theme-aware (light/dark), and print-friendly.

## The four ideas that outlast any tool

1. **Context is everything** — put the right things on the desk.
2. **Role · Task · Context · Format** — the anatomy of a good prompt.
3. **Climb the ladder** — Chat → Projects → Cowork → Agents.
4. **Verify before you trust** — a citation you haven't read is not evidence.
